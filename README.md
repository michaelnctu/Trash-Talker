# Trash-Talker
---
##功能
*一個可以惹毛三種直接的人話語產生器
##環境建置與需求
*Node.js v12.18.2
*Express v4.17.1
*Express-Handlebars v5.1.0
*body-parser v1.19.0
##安裝與執行步驟
*下載專案至本機
*git clone https://github.com/michaelnctu/Trash-Talker.git
*前往專案資料夾
*cd words_generator
*安裝套件
*npm install
*啟動專案
*npm run dev
*請至 http://localhost:3000 開始使用程式
